⚙️ Sambungkan Firebase dahulu

  1. Pergi ke console.firebase.google.com → cipta projek baru (percuma)
  2. Buka Build → Firestore Database → Create database → mula dalam mod test
  3. Buka Project settings → General → Add app → Web (</>) → salin firebaseConfig
  4. Tampal config tu dalam fail HTML ini, cari firebaseConfig = {...} di bahagian bawah
  5. Dalam Firestore Rules, tampal peraturan yang disediakan (rujuk mesej chat) supaya pasukan boleh baca/tulis

Selepas config diisi, muat semula halaman ini — banner ini akan hilang secara automatik.
